
# 🧠 AI Salary Dashboard

Sun'iy intellekt sohasidagi mutaxassislarning ish haqi (maosh) ma'lumotlarini chuqur tahlil qiluvchi interaktiv dashboard. Loyiha 15,000+ yozuvdan iborat ma'lumotlar bazasi asosida tayyorlangan va lavozim, tajriba darajasi, masofadan ishlash, geografiya va sanoat sohalari bo'yicha keng qamrovli tahlilni taqdim etadi.

## 📊 Dashboard haqida

Dashboard 6 ta asosiy bo'limdan iborat:

### 1. Overview – Executive Summary
Umumiy ko'rsatkichlar: o'rtacha maosh ($115K), median maosh ($100K), maksimal ($399K) va minimal ($33K) maoshlar, jami yozuvlar soni (15K). Eng yuqori maoshli lavozimlar reytingi va tajriba darajasi bo'yicha taqsimot.

Lavozimlar bo'yicha tahlil: noyob lavozim sarlavhalari, oylar bo'yicha dinamika, bandlik turi bo'yicha o'rtacha maosh va lavozim nomlari bo'yicha to'liq jadval.

## 🛠️ Texnologiyalar

- **Vizualizatsiya:** Power BI / Tableau (interaktiv dashboard)
- **Ma'lumotlar:** CSV / Excel formatdagi AI mutaxassislari maoshi datasetlari
- **Filtrlar:** Sana, davlat, sanoat, lavozim bo'yicha dinamik filtrlash
